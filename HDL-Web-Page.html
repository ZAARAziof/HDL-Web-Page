üî¢ Binary & Logic Gates

What is a truth table?
A truth table lists all possible input combinations and their corresponding outputs for a logic gate or circuit.
Explain the difference between AND, OR, and XOR gates.
AND outputs 1 if both inputs are 1; OR outputs 1 if at least one input is 1; XOR outputs 1 if exactly one input is 1.
How do you convert decimal 13 to binary?
13 in binary = 1101 (8 + 4 + 0 + 1).

üß¨ HDL & Circuit Design

Write HDL code for a two-input NAND gate.
CHIP Nand { IN a,b; OUT out; PARTS: Not(in=And(a=a,b=b), out=out); }
Write HDL code for a two-input AND gate.
CHIP And { IN a,b; OUT out; PARTS: And(a=a,b=b,out=out); }
Write HDL code for a Half Adder.
CHIP HalfAdder { IN a,b; OUT sum,carry; PARTS: Xor(a=a,b=b,out=sum); And(a=a,b=b,out=carry); }
Write HDL code for a Full Adder.
CHIP FullAdder { IN a,b,cin; OUT sum,cout; PARTS: HalfAdder(a=a,b=b,sum=ha1,sum=ha1sum,carry=ha1carry); HalfAdder(a=ha1sum,b=cin,sum=sum,carry=ha2carry); Or(a=ha1carry,b=ha2carry,out=cout); }
Write HDL code for a 2:1 Multiplexer.
CHIP Mux2 { IN a,b,sel; OUT out; PARTS: And(a=sel,b=b,out=and1); Not(in=sel,out=nsel); And(a=nsel,b=a,out=and2); Or(a=and1,b=and2,out=out); }
What is a D flip-flop in HDL?
A D flip-flop is a memory element that captures input D on a clock edge and holds it until the next clock. Example implementation uses a register clocked by clk.

üíæ CPU & Registers

What is the purpose of the Program Counter (PC)?
PC holds the address of the next instruction to be fetched from memory.
Explain the role of general-purpose registers.
General-purpose registers store data and addresses for immediate use by the CPU.

üíΩ RAM & Memory

What is volatile memory?
Volatile memory (like RAM) loses its data when power is turned off.
Difference between SRAM and DRAM?
SRAM uses flip-flops and is faster, DRAM uses capacitors and needs refresh cycles.
